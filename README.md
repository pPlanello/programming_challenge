## Requeriments
   1. Install node
   2. Install typescript
   3. Install java
   4. Install python
   5. Install JetBrains toolbox
   6. Install IntelliJ IDEA
   7. Install PyCharm

## How to use
### TypeScript

```
tsc && node .\out\solution.js
```

### Python

```
python ./python/solution.py
```

### Java

```
java ./java/src/main.java
```
